## 기억할 것 3가지

1. **Merge = 발행.** Merge 전에는 사이트에 반영되지 않는다.
2. 글·설정 수정은 GitHub에서 직접 편집하지 말고 Claude에게 시킬 것 (공백 하나로 빌드가 깨질 수 있음).
3. 새 회수 사례가 생기면 Claude에게 몇 줄로 보낼 것 → `docs/case-bank.md`에 등록되어 글 재료가 된다.
