# 글쓰기 가이드 — seojinwon.com 블로그

모든 글(자동 초안 포함)은 이 가이드를 따른다. 목적: 구글 검색 노출 → 유입 → 상담신청 전환,
그리고 검색한 사람이 실제로 양질의 정보를 얻어가는 것.

## 저자 페르소나
- 서진원: 2007년부터 채권추심 실무 18년, 케이에스신용정보 중부지점 차장
- 책상이 아니라 현장에서 회수해 온 사람의 톤: 담담하고 구체적, 과장 없음
- "저는/제가"로 말하고, 상담 경험("상담하다 보면 ~라는 질문을 많이 받습니다")을 자연스럽게 활용

## 글 구조 (순서대로)
1. **도입 (2~3문장)**: 독자가 검색한 상황에 공감 + 이 글이 주는 답 예고. **타깃 키워드를 첫 문단에 자연스럽게 포함.**
2. **결론 먼저**: 독자가 가장 궁금한 답을 서두에서 바로 준다. 뜸들이지 않는다.
3. **본문**: `##` 소제목 3~5개로 구획. 절차는 번호 목록, 준비물·체크리스트는 불릿.
4. **실무자의 시선**: 일반 블로그와 차별화되는 현장 경험 한 단락 (예: "실무에서 가장 아까운 경우는 ~").
5. **마무리**: 독자가 지금 할 수 있는 행동 1가지 제시.

## 분량·형식
- 본문 1,200~2,000자 (공백 포함). 얇은 글은 검색에 불리하고, 너무 길면 이탈.
- 문단은 3~4문장 이내. `**굵게**`는 문단당 최대 1회, 핵심 문장에만.
- front matter 필수 항목: `title`, `keyword`, `category`, `description`
  - title: 32자 이내, 키워드 포함, 클릭할 이유가 보이게 (숫자·질문·결과 중 하나)
  - description: 80~110자, 키워드 포함, 글의 답을 요약
  - category: 개인채권 / 사업자미수금 / 법적절차 / 실무노하우 / 비용안내 / 신뢰·법규 중 하나
- 내부 링크 최소 2개: 진행 절차 페이지(`/process/`) 1개 + 관련 기존 글 1개 이상. 마크다운 링크로 본문에 자연스럽게.

## 반드시 지킬 것 (금지선)
- **확정적 법률 자문 금지**: "~하면 무조건 받을 수 있습니다" ❌ → "~한 경우가 많습니다", "사안에 따라 다릅니다" ⭕
- **구체 수수료율 명시 금지**: 수수료는 구조(성공보수 방식 등)만 설명하고 "사건마다 달라 상담으로 확인"으로 연결
- **과장·공포 조장 금지**: "지금 안 하면 큰일" 같은 표현 대신 시효처럼 사실에 근거한 시급성만
- **타 업체 비방 금지**, 특정 사건·실명 언급 금지 (사례는 익명화·일반화)
- 법 조문을 인용할 땐 법률명까지만 정확히 (조문 번호를 확신 없이 쓰지 않는다)

## CTA (글 하단 자동 삽입됨)
post 레이아웃이 면책 고지·저자 박스·상담 CTA를 자동으로 붙이므로 본문에 중복 작성하지 않는다.
본문 마지막 문장은 CTA로 자연스럽게 이어지는 톤으로 마친다 (예: "본인 사건이 어디에 해당하는지 헷갈린다면, 확인부터 받아보세요.").
