# 구글 검색 노출·방문자 분석 설정 가이드 (사장님용)

이 문서는 **구글 서치 콘솔(Search Console)**과 **구글 애널리틱스(GA4)**를 등록하는 방법입니다.
한 번만 해두면 이후에는 자동으로 작동합니다. 두 작업 모두 10분 안에 끝납니다.

- **서치 콘솔** = 내 사이트가 구글 검색에 잘 나오는지 확인하고, 새 글을 빨리 색인시키는 도구
- **GA4** = 누가 어떤 글로 들어와서 상담신청까지 했는지 보는 도구

---

## 1부. 구글 서치 콘솔 등록

1. https://search.google.com/search-console 에 접속해서 구글 계정으로 로그인합니다.
2. "속성 추가"를 누르고, 오른쪽의 **URL 접두어** 칸에 `https://seojinwon.com` 을 입력합니다.
3. 소유권 확인 방법 중 **"HTML 태그"**를 선택합니다.
   - `<meta name="google-site-verification" content="여기에긴문자열" />` 같은 코드가 보입니다.
   - 이 중에서 **content="..." 안의 긴 문자열만** 복사해 두세요.
4. 그 문자열을 저(Claude)에게 알려주시거나, 직접 하시려면:
   - GitHub에서 `_config.yml` 파일을 열어 편집(연필 아이콘)
   - `# google_site_verification: 값` 줄에서 맨 앞의 `# `를 지우고 `값` 자리에 복사한 문자열을 붙여넣기
   - 초록색 "Commit changes" 버튼으로 저장
5. 5분쯤 기다렸다가 서치 콘솔로 돌아가 "확인" 버튼을 누릅니다. → "소유권이 확인됨"이 뜨면 성공.
6. 왼쪽 메뉴의 **Sitemaps**에 들어가 `sitemap.xml` 이라고 입력하고 제출합니다.
   - 이걸 해야 구글이 우리 사이트의 모든 글을 빠짐없이 찾아갑니다.

### 새 글을 빨리 색인시키는 법 (선택)
글 발행 후 서치 콘솔 상단 검색창에 글 주소(예: `https://seojinwon.com/blog/payment-order-guide/`)를
붙여넣고 → "색인 생성 요청"을 누르면 며칠 걸릴 색인이 보통 몇 시간으로 단축됩니다.

## 2부. 구글 애널리틱스(GA4) 등록

1. https://analytics.google.com 에 접속해 구글 계정으로 로그인합니다.
2. "측정 시작" → 계정 이름(예: 서진원) 입력 → 다음
3. 속성 이름(예: seojinwon.com) 입력, 시간대는 **대한민국**, 통화는 **원** 선택 → 다음
4. 업종·규모는 대충 선택해도 됩니다 → 만들기
5. 플랫폼 선택에서 **웹**을 누르고, 웹사이트 URL에 `https://seojinwon.com` 입력
6. 만들어지면 **측정 ID**가 보입니다. `G-` 로 시작하는 코드입니다 (예: `G-AB12CD34EF`)
7. 이 측정 ID를 저에게 알려주시거나, 직접 하시려면:
   - `_config.yml`에서 `# google_analytics: G-XXXXXXXXXX` 줄의 `# `를 지우고 실제 ID로 교체 → 저장

설정 후 하루 이틀 지나면 애널리틱스에서 방문자 수, 어떤 글로 들어왔는지, 상담 폼이 있는
메인 페이지로 얼마나 이동했는지를 볼 수 있습니다.
