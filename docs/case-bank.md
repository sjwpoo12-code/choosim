# 사례 은행 — 블로그 재구성용 실제 사례 모음

블로그 글에 넣을 실제 회수 사례의 원천 자료. 매일 초안 자동화가 글 주제와 맞는 사례를
골라 **재구성**해서 본문에 넣는다.

## 사용 규칙 (반드시 준수)

1. **익명화**: 의뢰인·채무자를 특정할 수 있는 정보(이름, 상호, 지역이 좁혀지는 표현, 정확한 날짜) 금지.
   업종·금액대·상황은 일반화해서 사용 (예: "식자재 납품업체" → "납품업체", "1억 2천" → "1억 원대").
2. **재구성 표기**: 사례 블록에는 반드시 "실제 사례를 각색·재구성한 것입니다" 문구를 포함한다.
3. **과장 금지**: 결과를 부풀리지 않는다. "전액 회수"는 실제 전액 회수된 사례에만 쓴다.
4. **회수 보장으로 읽히지 않게**: 사례 뒤에는 "사안마다 결과가 다르다"는 취지의 문장을 잇는다.
5. 한 사례는 여러 글에 재사용 가능하되, 같은 글에 2개 이상 넣지 않는다.

## 사례 목록

### CASE-01 · 거주불명 개인 채무자 — 현장 조사로 실거주지 파악
- 상황: 공사업 종사 개인 채무자, 주민등록상 거주불명으로 소재 파악 불가
- 조치: 현장 조사로 실거주지 직접 파악 → 유체동산 압류 집행 착수
- 결과: 원금 + 지연손해금 + 법적 조치 비용까지 전액 일시금 회수
- 어울리는 주제: 채무자 연락두절, 재산조사, 소재 파악, 떼인 돈 받는 법

### CASE-02 · 부실 법인 — 거래처 채권·공제조합 출자증권 압류
- 상황: 인테리어 공사업 법인이 공사대금 미지급, 법인 앞 재산 없어 보임
- 조치: 현장 방문으로 거래처 탐색 → 공사대금 채권 압류 → 건설공제조합 출자증권 추가 압류
- 결과: 채권 전액 일시금 회수
- 어울리는 주제: 사업자 미수금, 공사대금, 재산조사, 통장·채권 압류

### CASE-03 · 간판만 바꾼 회사 — 영업양수인 책임 추궁
- 상황: 납품대금(1억 원대) 미루던 거래처가 상호·대표 명의만 바꿔 같은 자리에서 영업 지속, 기존 회사 앞 재산 없음
- 조치: 세금계산서·입금계좌 대조로 신·구 회사의 실질적 동일성 입증 → 상호속용 영업양수인 책임 + 가족 명의 재산 이전 사해행위취소 검토 → 소송 후 새 회사 매출채권(정산금) 압류
- 결과: 회수 완료
- 어울리는 주제: 폐업한 거래처, 재산 은닉, 사해행위취소, 물품대금

### CASE-04 · 수년간 잠적한 채무자 — 공개 SNS 단서로 소재·재산 특정
- 상황: 제조업체 대표(채권자)가 오래 함께 일한 직원에게 대여금 등 3천만 원대 채권 보유.
  법적 책임 대신 공정증서(지연이자 약정)를 쓰고 분할 변제하기로 했으나 채무자가 곧 잠적.
  수년간 연락 두절, 주민등록상 주소도 위장전입이라 서류상 추적 불가. 주변에는 "돈을 잘 쓰고 다닌다"는 소문만.
- 조치: 유일한 단서였던 채무자의 공개 SNS 계정을 수일에 걸쳐 분석. 게시물에 노출된
  생활 반경 단서(방문 업소, 주거지에서 촬영된 풍경 등)를 교차 대조해 거주 지역을 좁히고
  실거주 아파트까지 특정. 게시물에 노출된 거래 은행을 근거로 임대차보증금·예금에 압류 신청 후 접촉.
- 결과: 지연이자 포함 원리금이 원금의 2배에 달했으나, 일시 변제 조건으로 3,500만 원에 회수 종결(현금 일시 수령).
- 어울리는 주제: 채무자 연락두절, 재산조사, 소재 파악, 빌려준 돈 받는 법, 공정증서
- 사용 시 주의: 조사 기법의 세부(어떤 업소·사진으로 특정했는지)는 흐리게 쓰고,
  "공개된 정보를 단서로 합법적 조사로 소재를 특정했다" 수준으로만 서술할 것.

글에 넣을 때 제가 다듬고 익명화합니다.

```
### CASE-XX · 한 줄 제목
- 상황: (어떤 채권, 어떤 채무자, 뭐가 문제였나)
- 조치: (무엇을 어떤 순서로 했나)
- 결과: (얼마나, 어떻게 회수됐나 / 실패했다면 왜)
- 어울리는 주제: (관련 키워드)
```
